# Touche Pas Au Klaxon

# C'est quoi ?

J'ai développé cette application web dans le cadre de ma formation. Il s'agit d'un site intranet de covoiturage pour une entreprise qui possède plusieurs agences (villes). Les employés peuvent proposer des trajets et consulter ceux des autres pour éviter de voyager seuls.

# Ce que j'ai utilisé

- PHP 8.2 avec une architecture MVC
- MySQL pour la base de données
- Bootstrap 5 et Sass pour le design
- Le routeur izniburak/router
- PHPStan pour vérifier la qualité du code
- PHPUnit pour les tests unitaires

# Comment installer le projet

# Ce qu'il faut avoir installé
- XAMPP (Apache + MySQL + PHP)
- Composer
- Node.js et npm
